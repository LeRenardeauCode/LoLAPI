# Projet Data Dragon — Interface React

## À propos du projet

Ce projet a été réalisé dans le cadre d’un travail scolaire et vise à explorer et afficher les données issues de **Data Dragon** (le service de données statiques de *League of Legends*).  
L’objectif est de manipuler des données JSON et de les présenter à l’utilisateur via une interface web moderne construite avec **React**.

Aucune clé API n’est nécessaire : toutes les données utilisées proviennent de **fichiers statiques** récupérés directement depuis Data Dragon ou d’un **fichier local `.json`** créé pour représenter les différentes régions du jeu.

---

## Technologies utilisées

- **React** — Librairie principale pour la construction de l’interface  
-- **Node** - Utilisation de Node pour le terminal et le lancement de projet
- **Vite** — Environnement de développement rapide et moderne  
- **npm** — Gestionnaire de dépendances  
- **React Router DOM** — Gestion de la navigation et des routes  
- **MUI (Material UI)** — Composants stylisés et cohérents pour un design moderne  
- **JavaScript (ES6)** — Langage principal du projet

## Installation et lancement

1. **Cloner le dépôt**

git clone https://github.com/<ton-utilisateur>/<nom-du-projet>.git
cd <nom-du-projet>

2. **Installer les dépendances**

npm install

3. **Lancer le serveur de développement**

npm run dev

4. **Accéder à l’application**

Rendez-vous sur http://localhost:5173 (ou le port affiché dans le terminal).
